# Flat CMS

データベース不要のPHP製CMS。JSONファイルでコンテンツを管理します。

## 特徴

- **データベース不要** — JSONファイルだけで動作
- **管理画面付き** — 記事・ページ・SEO・デザインをGUIで管理
- **シンプルな構成** — PHPとApacheだけあれば動く
- **レンタルサーバー対応** — FTPでアップするだけでOK

## 動作環境

- PHP 8.0 以上（推奨 8.2）
- Apache（mod_rewrite 有効）

## インストール

1. `src/` フォルダの中身をサーバーの公開ディレクトリにアップロード
2. ブラウザで `https://yourdomain.com/admin/` にアクセス
3. 初期設定画面でユーザー名とパスワードを設定

> **書き込みエラーが出た場合:** `data/` フォルダのパーミッションを `707` に設定してください。

## ローカル開発（Docker）

```bash
docker compose up
```

ブラウザで `http://localhost:8080` にアクセス。
